# FireChat 💬

FireChat is a real-time chat web application where users can create an account,
log in, and exchange messages and files instantly with other users.
Built with React and TypeScript on the frontend, and Firebase as the
backend-as-a-service for authentication, data, storage, and hosting.

🔗 **Live Demo:** [https://firechat-2026-646ad.web.app/](https://firechat-2026-646ad.web.app/)

## Features
- Real-time messaging
- User authentication
- File/image sharing

## Tech Stack
React · TypeScript · Firebase Auth · Firestore · Firebase Storage · Firebase Hosting

## Getting Started
npm install
npm run dev
